Stripe Navbar Clone 

This project recreates the Stripe-style navbar with a fully working mega menu for the “Solutions” section using React and JavaScript.
The focus is on clean UI, layout accuracy, responsive design, and real-world navigation behavior.

Tech Stack

React

JavaScript

CSS


Vite

Features
Desktop Navbar


Folder Structure
```
src/
  components/
    Navbar/
      Navbar.jsx
      SolutionsMegaMenu.jsx
      MobileMenu.jsx
  data/
    navbarData.js
  styles/
    index.css
    navbar.css
  App.jsx
  main.jsx

```
Installation & Run
Install Dependencies
```
npm install
```
Start Project
```
npm run dev

```

Then open in browser:
```
http://localhost:5173
```
Accessibility

Tab navigation supported

Escape to close menu
